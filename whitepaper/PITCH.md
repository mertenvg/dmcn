# DMCN — Elevator Pitch

Email is structurally broken. Spam dominates global traffic, business email compromise costs billions annually, and phishing remains the leading ransomware vector — not because these problems are unsolvable, but because every solution deployed over four decades has treated them as filtering problems rather than identity problems. The root cause is architectural: SMTP was designed in 1982 for a network of trusted academic nodes and provides no mechanism for verifying who is actually sending a message. No amount of filtering resolves an identity problem.

The Decentralized Mesh Communication Network (DMCN) fixes the root cause. Every participant has a cryptographic identity; every message must carry a valid signature from a registered identity before any relay node will accept it. Unsigned messages are rejected at the network boundary, forged messages are mathematically impossible, and the economics of spam collapse. The system is peer-to-peer, end-to-end encrypted, and — critically — designed so that all cryptographic complexity is invisible to users. A bridge architecture maintains interoperability with the four billion people on legacy email, and address portability means no one abandons their existing address to participate. Passkey technology has already proven that public-key cryptography can be made invisible to mainstream users at scale. The DMCN applies that proof to email.
